# automatic_tollcollection
Automatic Toll Collection using RFID 

### Hardware Requirements

    1> RF ID reder
    2> RF ID cards(you have minimum 4 to 5 cards)
    3> Microcontroller AT89c52
    4> EEPROM IC
    5> RTC(DS1307)
    6> 20*4 LCD Display
    7> RS232 cable
    
### Software Requirements

    1> Keil5
